# Teste DISC de Personalidade

Um teste de personalidade baseado no modelo DISC, que avalia quatro perfis comportamentais:

- **D** - Dominância
- **I** - Influência
- **S** - Estabilidade
- **C** - Conformidade

## 🚀 Acesse o Teste

Experimente agora: [https://gabrielrezende079.github.io/Teste-DISC-de-Personalidade/](https://gabrielrezende079.github.io/Teste-DISC-de-Personalidade/)

## 📖 Sobre o Modelo DISC

O modelo DISC é uma ferramenta utilizada para análise de comportamento e perfis de personalidade. Ele classifica as pessoas em quatro quadrantes, ajudando a entender melhor como cada indivíduo age, se comunica e toma decisões.

- `TesteDISC.csv` - Banco de dados com as questões e pontuações do teste
